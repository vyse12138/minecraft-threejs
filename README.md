# minecraft-threejs

[English](https://github.com/Vyse12138/minecraft-threejs/blob/main/README_EN.md) ｜ 中文

基于 Three.js 和 TypeScript 的网页版我的世界

试玩：[Minecraft - Three.js](https://mc-zh.yuleiz.com/)

建议使用 Chrome 来获得最佳的体验

## 已支持:

- 方块的放置 / 破坏
- 鼠标滚轮（数字键）选择不同的方块类型
- 移动和碰撞检测
- 随机的地形和树木生成
- 无限的世界
- 保存 / 读取游戏
- 音效和背景音乐
- 可调节的渲染距离和视野范围
- 高亮准心方块
- 基本的 UI
